# Schémas XSD TEIF — archive de versions

TTN ne maintient pas d'archive publique des anciennes versions du schéma XSD : seule la version en cours (bêta ou stable) est téléchargeable sur [tradenet.com.tn/elfatoora.html](https://www.tradenet.com.tn/elfatoora.html). Ce dossier comble ce manque en conservant les versions successives, pour permettre aux intégrateurs de comparer, migrer et diagnostiquer sans dépendre de ce qui est disponible *aujourd'hui* sur le site officiel.

## Versions disponibles

| Fichier | Version | Statut | Récupéré le |
|---|---|---|---|
| [`elfatoora-1.8.9.xsd`](elfatoora-1.8.9.xsd) | 1.8.9 | **Production** (en vigueur) | 2026-07-11 |
